MCP для Cursor и Claude

MCP (Model Context Protocol) — способ подключить API СамОтправил к AI-ассистентам в IDE. Сервер даёт поиск по документации, 58 typed tools (включая parity с Python SDK), 9 MCP resources, offline-fallback и безопасные пресеты (READ_ONLY, dry_run).

  • GitHub — samotpravil-mcp

  • npm — samotpravil-mcp (актуальная версия: 1.8.0+)

  • MCP Registry — io.github.dkanster/samotpravil-api-mcp

  • Примеры конфигурации и сценарии

  • Python SDK — те же методы доступны как MCP tools (v1.3+)

Установка в Cursor

Файл .cursor/mcp.json в корне проекта:

{
  "mcpServers": {
    "samotpravil": {
      "command": "npx",
      "args": ["-y", "samotpravil-mcp@latest"],
      "env": {
        "SAMOTPRAVIL_API_KEY": "ваш_api_ключ",
        "SAMOTPRAVIL_READ_ONLY": "1",
        "SAMOTPRAVIL_ALLOW_SEND": "0"
      }
    }
  }
}

Ключ API — в заголовке Authorization (см. раздел «Авторизация»). Для только документации ключ можно не указывать. После правок: Settings → MCP → Reload.

Maintainer'ам документации: при POSTMAN_API_KEY доступны tools postman_sync_snapshot, postman_diff_snapshot для синхронизации коллекции со snapshot в репозитории.

Также поддерживаются Claude Desktop и VS Code Copilot — см. docs/EXAMPLES.md.
